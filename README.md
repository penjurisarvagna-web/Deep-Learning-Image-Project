# Deep Learning Image Classification Project

## Objective
To build an image classification model using TensorFlow and the CIFAR-10 dataset.

## Dataset
CIFAR-10 Dataset

## Model
Convolutional Neural Network (CNN)

## Results
The model was trained for 5 epochs and achieved good test accuracy on the CIFAR-10 dataset.

## Technologies Used
- Python
- TensorFlow
- Keras
- Google Colab
